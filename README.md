# Thirsty4DnD Assets

This repository contains the artwork and media assets used across the **Thirsty4DnD** project.

## Contents

- Campaign cover art
- Battle maps
- NPC portraits
- Creature artwork
- Magic items
- Handouts
- Tavern and location artwork
- Logos and branding
- Icons and UI assets

These assets are primarily hosted here to provide stable image links for use in Homebrewery, Patreon, virtual tabletops, and other Thirsty4DnD content.
